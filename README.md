# Pharmacy Inventory System

A simple Java program to manage pharmacy stock (medicines and medical devices).

Files
Item.java - Base class for all products

Medication.java - Medicine with expiration date

MedicalDevices.java - Equipment with warranty period

Inventory.java - Stores and displays items

Main.java - Runs the program

## How to Compile & Run

 Prerequisites
- Java Development Kit (JDK) 8 or higher
- Command line / terminal

Steps

1. Compile all Java files:
   javac *.java

2. Run the application : 
   java Main
